# Employee-Payroll-SQL-Project
Designed and implemented a self-learning SQL project focused on employee payroll data analysis. 
Created relational database tables, inserted sample data, and wrote SQL queries to retrieve meaningful insights. 
Practiced core SQL concepts including SELECT, WHERE, JOIN, GROUP BY, HAVING, ORDER BY, and aggregate functions such as COUNT, AVG

## Project Overview
This is a self-learning SQL project created to understand and practice
SQL query writing. The project focuses on employee payroll data analysis
using core SQL concepts.

## Technologies Used
- SQL
- SQL Server

## Key Concepts Practiced
- SELECT, WHERE
- JOIN
- GROUP BY, HAVING
- ORDER BY
- Aggregate functions (COUNT, AVG, SUM)

## Learning Outcome
This project helped me gain confidence in writing SQL queries and
understanding database relationships.
